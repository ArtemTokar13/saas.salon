# Flutter App Assets

Place your app assets in the following directories:

## images/
- App logo
- Placeholder images
- Company/salon images
- Staff photos

## icons/
- Custom app icons
- Service icons
- Feature icons

## Recommended sizes:
- Logo: 512x512px (PNG)
- Icons: 48x48px, 96x96px (PNG)
- Images: Various sizes with aspect ratio preserved

Remember to update `pubspec.yaml` if you add new asset directories.
